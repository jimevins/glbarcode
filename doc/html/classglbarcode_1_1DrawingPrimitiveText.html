Detailed Description

A character string drawing primitive. 

@ =  origin (x,y) from top left corner of barcode

         ____        _  ------------------
        /    \      | |                  ^
       /  /\  \     | |                  |
      /  /__\  \    | |___     ____      |
     /  ______  \   | ._  \   /  __|     | ~fsize
    /  /      \  \  | |_)  | |  (__      |
   /__/        \__\ |_.___/   \____|     |
                                         v
                      @ ------------------
                      x = horizontal center

All units are in points ( 1 point = 1/72 inch ).
